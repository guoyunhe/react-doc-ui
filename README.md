# react-doc-ui

## Install

```bash
npm i react-doc-ui
```

## Usage

```jsx
import cn from 'classnames';

function App() {
  return <div className={cn('foo', 'bar')}>foobar</div>;
}
```

```jsx
import cn from 'classnames';

function foobar() {
  return 'foobar';
}

function App() {
  return <div className={cn('foo', 'bar')}>foobar</div>;
}

render(<App />);
```
